Description

Модуль для определения типов данных, связанных с метаданными книг.

Модуль BookTypes содержит определения классов и типов данных для представления метаданных о книгах и их сериальных частях. Он также предоставляет методы для работы с обложками, сериями и другими аспектами книг.
